# Our Cookbook

Emily and Shingai's family recipe cookbook.

## Quick Start

```bash
npm install
npm run dev    # starts at localhost:4323
```

## Add a Recipe

Create a markdown file in `src/content/recipes/` — or just tell Claude what you want to cook:

> "Add a lemon garlic chicken recipe, 30 minutes, serves 4"

## Built With

- [Astro](https://astro.build) — fast static site builder
- [Tailwind CSS](https://tailwindcss.com) — utility-first styling
- [Claude Code](https://claude.ai/code) — AI development companion
